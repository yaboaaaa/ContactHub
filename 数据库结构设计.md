# 第3章 详细设计

## 3.1 数据库设计

### 数据库结构设计

#### 用户信息表（users）

| 字段名称 | 数据类型 | 备注 |
| :--- | :--- | :--- |
| id | BIGINT | 用户编号（主键） |
| username | VARCHAR(50) | 用户名 |
| password | VARCHAR(200) | 密码 |
| email | VARCHAR(100) | 用户邮箱 |
| role | VARCHAR(20) | 用户角色 |
| enabled | BOOLEAN | 是否启用 |
| avatar_url | VARCHAR(500) | 头像地址 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

#### 联系人分组表（contact_groups）

| 字段名称 | 数据类型 | 备注 |
| :--- | :--- | :--- |
| id | BIGINT | 分组编号（主键） |
| name | VARCHAR(50) | 分组名称 |
| user_id | BIGINT | 用户编号（外键） |
| is_default | BOOLEAN | 是否默认分组 |
| sort_order | INTEGER | 排序顺序 |
| created_at | DATETIME | 创建时间 |

#### 联系人信息表（contacts）

| 字段名称 | 数据类型 | 备注 |
| :--- | :--- | :--- |
| id | BIGINT | 联系人编号（主键） |
| user_id | BIGINT | 用户编号（外键） |
| group_id | BIGINT | 分组编号（外键） |
| name | VARCHAR(50) | 姓名 |
| family_name | VARCHAR(30) | 姓 |
| given_name | VARCHAR(30) | 名 |
| uid | VARCHAR(255) | 唯一标识 |
| gender | INTEGER | 性别 |
| phone_mobile | VARCHAR(20) | 手机 |
| phone_home | VARCHAR(20) | 家庭电话 |
| phone_work | VARCHAR(20) | 工作电话 |
| email | VARCHAR(100) | 邮箱 |
| company | VARCHAR(100) | 公司 |
| job_title | VARCHAR(50) | 职位 |
| province | VARCHAR(30) | 省份 |
| city | VARCHAR(30) | 城市 |
| district | VARCHAR(30) | 区县 |
| address_detail | VARCHAR(200) | 详细地址 |
| birthday | DATE | 生日 |
| notes | VARCHAR(500) | 备注 |
| avatar_url | VARCHAR(500) | 头像地址 |
| is_deleted | BOOLEAN | 是否已删除 |
| deleted_at | DATETIME | 删除时间 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |
